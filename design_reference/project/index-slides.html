The design system
Motivating, but never overwhelming.
Taskko's design system balances reward-loop energy with focus-friendly calm. Every choice (colour, type, motion) is grounded in an HCI rationale aimed at the student burnout problem.
Colour palette
Sky blue + warm peach
Cool sky blue (#1FB6F0) for focus surfaces; warm peach (#FF8A65) reserved for streaks/energy. Low-saturation mint background reduces visual fatigue during long sessions.
Typography
Fraunces + Manrope + Mono
A warm display serif (Fraunces) for emotional headlines, a clean humanist sans (Manrope) for UI legibility, JetBrains Mono for numeric stats (streaks, points, timers).
Gamification
Rewards, sandboxed
Points, ranks, badges and squads live in the Hub tab. They surface as celebration moments on the dashboard but never crowd out the next-task action.
Motion & feedback
Confirm every win
Tasks animate to green on completion. The mood picker rewrites the next session instantly. Tako (the AI) has bouncing-dot typing — feedback is the default.

SYSTEM
01 · First launch
Splash
Cold-start screen. Builds brand recall while assets load. Holds the user briefly so the brand registers before any UI work begins.
Recognition over recall
The mark is a checkmark in a squircle + a peach spark — the verb (mark done) and the streak motif are encoded in the brand itself.
Colour & emotion
Cool sky-blue gradient communicates calm focus; warm peach spark balances it with energy. The two product pillars in one sigil.
Feedback (Nielsen #1)
Shimmer bar + "loading…" label tell the user the system is alive — no silent waits.
Constraints
No tappable controls; prevents action before the app is ready. Auto-advances on a predictable clock.

02 · Onboarding (3 slides)
Plan → Streaks → Squad
Three illustrated value-prop slides. Skip is on every slide; the last CTA changes from "Next" to "Get started" to close the arc.
Affordance
"Next" CTA is the brightest element. "Skip" is muted in the corner — both visible, hierarchy clear.
Escape hatch (Nielsen #3)
Skip is on every slide. Users are never trapped — exits cleanly to Sign-up.
Chunking (Miller's law)
One idea per slide + one vignette illustration. Cognitive load minimised.
Recognition
Slides preview real UI fragments (task cards, badge, leaderboard) so they're familiar when users meet them later.
Closure (Gestalt)
Final CTA flips from "Next" to "Get started" — visual + verbal close on the onboarding arc.

03 · Auth
Sign up
Minimum-viable form: 3 fields + agree. Social options up top, real-time validation, password strength meter.
Constraints
Real-time validation catches bad emails, short passwords, missing terms — errors prevented, not punished after submit (Norman).
Feedback
Password strength bar fills as user types. Per-field error messages are specific, not generic "invalid".
Visibility
Show/Hide password toggle lets users verify input — anti-frustration pattern.
Affordance
Primary CTA is greyed and disabled until form is valid — it looks unclickable because it IS unclickable.
Gulf of Execution
One clear forward path. No optional fields to dither over.

04 · Auth
Log in
Returning-user path. Bigger logo for instant recognition; re-engagement copy primes the variable reward.
Recognition over recall
Large logo at top — returning users see the brand instantly, confirming they're in the right place.
Proximity (Gestalt)
"Forgot?" sits inline with the password field it relates to, not in a hidden menu.
Emotion (Hook model)
"Your streak's waiting" is a variable-reward cue — re-engagement built into the gate.
Constraints
CTA disabled until both fields look valid. No submit-and-shame loop.

05 · Core
Home / Dashboard
Pull-of-the-day surface. Surfaces exactly one obvious next action. Stats above the fold answer "how am I doing?" in under a second.
Visibility
Streak, rank, mood, next task all visible without scroll — no menus needed.
Mapping
Mood emoji directly drives session length and break ratio. Input maps to output.
Gulf of Execution
Hero "Next up" card states verb + concrete task — students never wonder what to start.
Gulf of Evaluation
Progress bars on streak (5/5) + rank (68% to Elite) make state instantly readable.
Visual hierarchy
CTA uses the most saturated colour. Stats use medium contrast. Tasks neutral.
Colour & burnout
Warm peach = energy. Cool sky-blue = focus. Low-sat mint background reduces eye fatigue.

06 · Core
AI Task Breakdown
Bridges the hardest student problem: "where do I even start?" Takes a big, vague goal and produces a reviewable, editable plan.
Affordance
Textbox looks like a textbox. "Break it down" button has gradient + lift shadow — pressable.
Constraints
Sample-goal chips bound the AI input space without restricting freedom. Prevents junk prompts upstream.
Feedback
4-step parsing indicator during generation. Each step lights up as it completes.
Mapping
Top progress indicator matches actual flow position (Goal → Break down → Customize).
Closure (Gestalt)
Numbered identical cards form a clear list gestalt. Total time/points footer closes the loop.
User control
AI suggests; user decides — every task is editable, deletable, regenerable.

07 · Core
Gamification & Social Hub
Balances dopamine (badges, leaderboard) with reflection (report card). One-tap sharing turns wins into accountability + organic growth.
Visibility
Newly unlocked badge surfaces at top with an immediate Share affordance. Celebration is instant.
Similarity (Gestalt)
All badges share an octagonal medal shape; only colour/saturation differentiate earned vs locked.
Hierarchy (leaderboard)
"You" row uses primary accent + bolder weight — findable in ≤1 second.
Closure
Weekly report card summarises a week into 4 stats + a sparkline — a complete, shareable unit.
Mapping
Share button produces an actual preview of what will post — WYSIWYG.
Avoiding overwhelm
Gamification is scoped to its own tab; never leaks into productivity surfaces.

08 · Core
AI Chatbot — Tako
Conversational interface for emotional support + re-planning. Designed to feel like a friend, not a tool. Mood-aware avatar matches user state.
Proximity (Gestalt)
Same-speaker messages are spatially grouped. Avatar appears once per block, not per message.
Continuity
Bubbles flow top-down in time order. New messages animate in, anchoring the eye.
Colour coding
User = primary blue. Tako = white with subtle border. Dialogue readable at a glance.
Affordance
Quick-prompt chips above the input remove the cold-start problem.
Feedback
Typing indicator with bouncing dots — no silent waits.
Nudge separation
High-priority nudge cards render with distinct gradient + action buttons.
